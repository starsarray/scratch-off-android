# 刮卡工坊玩法文档

## 核心循环

1. 选择一张刮刮卡。
2. 支付该卡成本后开始本轮。
3. 刮开奖区，满足该卡规则则获得奖励。
4. 每刮完一张卡，该卡获得 1 点经验。
5. 卡等级提升后，该卡中奖率和高奖符号概率同步提升。
6. 用现金购买全局升级，继续推进更高阶卡。

## 财富与解锁

- 游戏存在两个数值：
- `现金`：当前可消费余额。
- `累计财富`：历史赚到的总财富，用于解锁新卡。
- 新卡解锁只看 `累计财富`，不要求当前手头现金达到阈值。

## 五种刮刮卡

### 1. 日常零活

- 结构：3格横排
- 成本：`$1`
- 规则：刮出 3 个相同符号即可获胜
- 解锁：初始开放
- 定位：新手入门卡，成本最低，收益平稳

### 2. 双倍成对

- 结构：2 x 2 配对
- 成本：`$10`
- 规则：任意一行出现一对相同符号即可获胜
- 解锁：累计财富达到 `150`
- 定位：节奏更快，适合前期滚雪球

### 3. 迷你连线

- 结构：5格连线
- 成本：`$100`
- 规则：任意连续 3 格相同即可获胜
- 解锁：累计财富达到 `1,200`
- 定位：中期过渡卡，开始强调连线收益

### 4. 果园丰收

- 结构：6格果园
- 成本：`$2,000`
- 规则：6格内任意符号出现 3 个及以上即可获胜
- 解锁：累计财富达到 `12,000`
- 定位：中后期卡，奖励跨度明显变大

### 5. 急速翻倍

- 结构：4格主区 + 1 个奖励位
- 成本：`$10,000`
- 规则：主区任意 3 个相同即中奖，奖励位有机会翻倍
- 解锁：累计财富达到 `60,000`
- 定位：高风险高回报卡，适合作为后期主力

## 卡等级与经验

- 每张卡独立记录：
- `等级`
- `经验`
- 每刮完一张卡，无论输赢，该卡都会获得 `1 点经验`
- 升级经验公式：
- `下一等级所需经验 = 3 + 当前等级 * 2`

### 卡等级效果

- 提高该卡基础中奖率
- 提高该卡高奖符号的出现权重
- 高阶卡等级拉高后，收益成长更明显

## 全局升级

### 1. 刮卡运气

- 作用对象：所有卡
- 效果：
- 提高整体中奖率
- 提高高奖符号出现概率
- 适合长期投资

### 2. 刮除范围

- 作用对象：刮卡半径
- 效果：
- 扩大每次擦除范围
- 减少来回补刮次数
- 手感更轻松

### 3. 铝币

- 作用对象：刮卡力度与效率
- 效果：
- 提高单次滑动的有效刮开效率
- 降低自动结算所需的刮开比例
- 等级越高，刮卡越顺手越快

## 升级花费

- 当前实现使用指数成长：
- `升级价格 = baseCost * 1.55 ^ 当前等级`

### 初始基础价格

- 刮卡运气：`$8,000`
- 刮除范围：`$2,500`
- 铝币：`$5,000`

## 数值方向

### 前期

- 优先投入 `刮除范围`
- 先保证操作效率和刮开手感

### 中期

- 把 `刮卡运气` 提到稳定档位
- 逐步切换到 `迷你连线` / `果园丰收`

### 后期

- 用高等级卡和高运气组合主刷 `急速翻倍`
- 再补 `铝币` 提升单位时间收益

## 后续可扩展方向

- 稀有事件卡
- 临时增益道具
- 多层刮区
- 连胜和保底系统
- 图鉴与成就系统
